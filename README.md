# MinePy (2D)

Простой и быстрый 2D-движок для игры в стиле Terraria, написанный на **Python 3.14** с использованием библиотеки **Arcade**.

## 🚀 Особенности
*   **Современный стек:** Полная поддержка Python 3.14.
*   **Производительность:** Работает на базе Arcade, что обеспечивает плавную отрисовку и высокую скорость обработки событий.
*   **Система инструментов:** Разные инструменты имеют уникальные характеристики (разная скорость копания и урон).
*   **Интерактивный инвентарь:** Удобная система Drag & Drop для управления предметами.
*   **Таймеры раскопки:** Реалистичная механика добычи ресурсов
