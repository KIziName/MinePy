# MinePy (Minecraft Clone)

Оптимизированный 3D-движок воксельного мира на базе Python и библиотеки Ursina Engine. Проект демонстрирует реализацию базовых механик воксельной графики, процедурной генерации ландшафта, физики объектов и управления состояниями игрока.

--------

## 🛠 Ключевые возможности

* **Процедурная генерация:** Динамическое создание ландшафта на основе шума Перлина (Perlin Noise) с генерацией руд и биомов.

* **Чанковая система рендеринга:** Эффективная загрузка и выгрузка секторов мира вокруг игрока для оптимизации расхода оперативной памяти.

* **Система сохранений (Persistence Layer):** Кеширование изменений мира и интеграция с системным каталогом приложений (`%APPDATA%/.minecraft/saves`).

* **Инвентарь и Хотбар:** Полный цикл взаимодействия с предметами — сбор выпавших объектов, перемещение по слотам и использование блоков.

* **Сущности и AI:** Модель поведения дружелюбных и враждебных мобов (патрулирование, преследование игрока, механика урона и взрывов).

* **Адаптивный UI/UX:** Динамическая подстройка графического интерфейса под различные разрешения экрана и полноэкранный режим.
