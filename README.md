# ⛏️ MinePy:(2D)

**MinePy** — это 2D песочница с процедурной генерацией мира, динамической сменой времени суток, крафтом и враждебными мобами. Проект полностью написан на Python с использованием библиотеки **PYgame**.

---

## ✨ Особенности игры

* **🌍 Процедурный мир:** Бесконечная генерация чанков с холмами, рудой (медь, железо, золото, уголь) и деревьями.
* **⚔️ Боевая система и мобы:**
  * **Днём:** Мирные и синие слаймы.
  * **Ночью:** Зомби, Летающие Глаза и Скелеты.
* **🛠️ Система инвентаря и крафта:** Удобный инвентарь (клавиша `E`), перетаскивание предметов мышкой (Drag & Drop) и меню создания предметов (мечи, кирки, факелы, зелья, броня).
* **💾 Сохранение мира:** Все изменения карты, инвентарь, мобы и позиция игрока автоматически сохраняются в папку `%AppData%/MinePy`.
* **⏱️ Оптимизация и Frame Pacing:** Стабильный лимит в **60 FPS** без скачков и фризов благодаря PYgame
* **🖥️ Интерфейс:** Поддержка полноэкранного и оконного режимов.

---

## 🎮 Управление

| Клавиша | Действие |
| **A / D** | Движение влево / вправо |
| Space  | Прыжок |
| **ЛКМ (Left Click)** | Ломать блоки / Атаковать мобов / Взаимодействовать |
| **ПКМ (Right Click)** | Ставить блоки |
| **E** | Открыть / Закрыть Инвентарь и Крафт |
| **1 - 0** | Быстрый выбор слота на хотбаре |
| **Esc** | Пауза-Меню

(Проект находится в альфа-версии могут быть баги и ошибки)
